Getting Started

Welcome to FontPair AI. This tool is designed to help professional designers and developers analyze typography, find perfect pairings, and ensure accessibility compliance using advanced AI.

To begin, you'll need to download the application for your operating system and set up your API key.

Go to Setup Guide →

API Key Setup

FontPair AI operates using your own Google Gemini API key. This ensures your data privacy and eliminates recurring subscription fees.

How to get a key

  1. Visit Google AI Studio.
  2. Click "Create API Key".
  3. Copy the key string (starts with AIza...).

Adding it to FontPair AI

Open the application, go to Settings > API Configuration, and paste your key into the input field. Click "Verify & Save" to confirm it's working.

Font Analysis

Our 14-point analysis engine breaks down any font file (.ttf, .otf) or image upload into actionable typographic data.

  • Classification: Identifies if the font is Serif, Sans, Slab, Display, etc.
  • X-Height: Measures the height of lowercase letters relative to uppercase.
  • Contrast: Analyzes the difference between thick and thin strokes.
  • Mood/Voice: AI-determined emotional characteristics (e.g., "Playful", "Corporate").

Simply drag and drop your file onto the dashboard to begin analysis.

Pairing Critique

The heart of FontPair AI. Select a primary heading font and a secondary body font to receive a detailed critique.

The AI assigns a score from 1-10 based on:

  • Visual Harmony: Do the shapes complement or clash?
  • Hierarchy: Is there enough distinction between the two?
  • Context Suitability: Is this pairing appropriate for your project type?

Glyph Comparison

Impostor shapes can ruin legibility. Our Glyph Comparison tool isolates and overlays commonly confused characters.

We automatically test:

  • I / l / 1 (Uppercase I, lowercase L, number 1)
  • 0 / O (Zero vs Uppercase O)
  • 6 / b and 9 / q

Use this feature to ensure your chosen font is safe for interface design and data-heavy applications.

Troubleshooting

API Key Errors

If you receive a 401 or 403 error, your API key may be expired or invalid. Check your quota at Google AI Studio. Remember the free tier has limits (15 req/min).

Font Not Loading

Ensure your file is a valid .ttf, .otf, or .woff2. Corrupted font files or unsupported formats (like older .eot) will be rejected.
